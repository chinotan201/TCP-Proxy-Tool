# Hexdump Tool in Python

This Python script generates a **hexadecimal dump** of binary or text data. It prints the data in a readable format, showing both the hexadecimal representation and the corresponding ASCII characters.

You can customize the output to be printed to the console or saved to a file.

## Features
- Generates a hexdump of the given input (binary or text).
- Displays the hexadecimal representation and the corresponding ASCII characters.
- Customizable output length (default is 16 bytes per line).
- Option to **save the hexdump to a file** or display it in the console.

## Requirements
- Python 3.x

## Installation
No installation is required. Simply download or clone this repository to your local machine.

### Example 1: Print hexdump to the console
